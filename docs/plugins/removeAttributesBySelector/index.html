removeAttributesBySelector

Removes specific attributes from elements that match a CSS selector.

Usage

  • Basic
  • Parameters
svgo.config.js
module.exports = {
  plugins: [
    "removeAttributesBySelector"
  ]
}
svgo.config.js
module.exports = {
  plugins: [
    {
      name: "removeAttributesBySelector",
      params: {
        selectors: null
      }
    }
  ]
}

Parameters

selectors

An array of objects with two properties, selector, and attributes, which represent a CSS selector and the attributes to remove respectively.

Demo

Live Editor
const svg = `
<?xml version="1.0" encoding="UTF-8"?>
<svg xmlns="http://www.w3.org/2000/svg" viewBox=" 0 0  150 100 " width="150">
  <!-- Created with love! -->
  <defs>
    <ellipse cx="50" cy="50.0" rx="50.00" ry="auto" fill="black" id="circle"/>
  </defs>
  <g>
    <use href="#circle" transform="skewX(16)"/>
    <rect id="useless" width="0" height="0" fill="#ff0000"/>
  </g>
</svg>
`;

const svgoConfig = {
  js2svg: { indent: 2, pretty: true },
  plugins: [
    "removeAttributesBySelector"
  ]
}

render(<SvgoPreview svg={svg} svgoConfig={svgoConfig}/>);
